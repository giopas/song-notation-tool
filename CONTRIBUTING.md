# Contributing to Song Notation Tool

Thanks for taking the time to look at this project! 🎸

> **Reminder:** This is a draft / demo project built through AI-assisted ("vibecoded") development. It works, but it's rough — contributions and feedback are warmly welcome.

---

## Reporting Bugs

Please use the **Bug Report** issue template when opening a bug.

Try to include:
- Your OS and Python version (`python3 --version`)
- What you were doing when the bug occurred
- What you expected to happen vs. what actually happened
- Any error messages from the terminal

---

## Suggesting Features

Use the **Feature Request** issue template.

Keep in mind this is a personal tool — not every feature will make sense to add — but all ideas will be considered!

---

## Code Contributions

At this stage there is no formal PR process. If you'd like to contribute code:
1. Fork the repo
2. Make your changes in a clearly named branch
3. Open a Pull Request with a short description of what you changed and why

Please keep changes focused — one feature or fix per PR makes review much easier.

---

## Style Notes

- Pure Python standard library only (no external dependencies)
- Tkinter for the UI
- Target: Python 3.8+ on macOS, Windows, and Linux
- Inline comments are encouraged — the code was written fast and can always use more clarity
